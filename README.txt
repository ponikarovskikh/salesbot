1. Добавить  проект в вашу директрию :
Linux: git clone https://github.com/ponikarovskikh/salesbot.git
2. Создать venv  вашей директории: python -m venv myenv
активировать source myenv/bin/activate
2. Загрузить зависимости  pip install requirements.txt
3. Добавить config.py в проект
3. Запустить бот через файл botcheck.py
